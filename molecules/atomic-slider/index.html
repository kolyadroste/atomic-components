Atomic Slider with responsive columns

An extra component is used to achieve this: atomic-slider-responsive-cols
This component reads the slider and wrap them into a wrapper slide.

By default bootstrap grid classes are placed. But these classes can be configured by using rowClass="row" colClass="col-" colClassAdditional="column"
